# Code Of Conduct

SilentCart aims to be a calm, respectful, technically serious project.

## Expected Behavior

- Be respectful in disagreements
- Critique ideas and implementations, not people
- Share context, benchmarks, and reasoning when challenging a decision
- Assume good faith, but stay precise
- Protect user privacy and security in both code and discussion

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks
- Doxxing or sharing private information without consent
- Publicly posting exploit details that put operators or users at risk
- Repeated bad-faith disruption, trolling, or hostile conduct
- Discriminatory language or behavior

## Scope

This applies to:

- issue discussions
- pull requests
- project chat spaces
- release notes and public support threads

## Enforcement

Project maintainers may remove comments, close discussions, reject contributions, or block participants whose behavior harms the project or its community.

## Reporting

If you experience or witness harmful conduct, contact the maintainer through the repository profile or project contact channel once the repository is public.
